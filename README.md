# demoApp

Sinatra API supporting registration & authentication

## API Operations

* `/api/login` - acquire an auth token
* `/api/register` - register a new account
* `/api/notes` - view notes (auth required)

ex: /api/notes


## Install App

* `bundle install`

## Start App

Either of the following should work: 

* `rackup` 
* `heroku local`
